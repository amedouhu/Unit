<h1 align="center">Unit</h1>

<p align="center">テキストメッセージの通信モデル</p>

## 概要

* Unitは、クライアントサーバシステム型のテキストメッセージの通信モデルです。
* 通信部分ではTCP/IPプロトコルの簡単な技術を使用しています。
* ※現在、Unitは通信モデルであるため根底部分のみを提供し、セキュリティなどの実用面でのサポートをしていません。
